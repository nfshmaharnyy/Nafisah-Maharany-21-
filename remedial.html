Pengertian HTML

"HTML atau HyperText Markup Language merupakan sebuah bahasa markah untuk membuat halaman web."

HTML itu adalah sebuah bahasa yang menggunakan markup atau penanda untuk membuat halaman web.

Penanda atau markup ini, nanti akan kita sebut dengan Tag.

HTML berperan untuk menentukan struktur konten dan tampilan dari sebuah web.

Perkembangan HTML

  • HTML punya beberapa versi, dari versi yang paling tua hingga yang terbaru. Berikut ini perkembangan versi HTML:

  • [Draft] HTML 1.0 (Juni 1993) adalah versi HTML pertama, namun tidak resmi dirilis;

  • HTML 2.0 (24 November 1995) adalah versi HTML kedua yang resmi pertama kali beredar di pasaran dan dirilis oleh IETF;

  • [Draft] HTML 3.0 (28 Maret 1995) versi ini gagal beredar, karena banyak perubahan yang memicu perdebatan;

  • HTML 3.2 (14 Januari 1997) versi resmi yang dirilis W3C pertama kali.

  • HTML 4.0 (24 April 1998) versi pengembangan dari yang sebelumnya;

  • HTML 4.01 (24 Desember 1999) versi perbaikan dari HTML 4.0;

  • XHTML 1.0 (26 Januari 2000) pengembangan dari HTML 4.01 dengan mengadopsi XML;

Membuat List pada HTML

elemen list digunakan untuk membuat daftar item

Dalam HTML, daftar (list) digunakan untuk menyusun elemen-elemen dalam bentuk urutan atau poin-poin

Ada tiga jenis list utama yaitu:

Unordered List ('ul'): Digunakan untuk membuat daftar yang tidak berurutan, biasanya ditandai dengan bullet points

Contoh:

  • Ayam
  • Bakso
  • Pangsit

Ordered List ('ol'): Digunakan untuk membuat daftar yang berurutan, biasanya ditandai dengan angka atau huruf

Contoh:

  1. Mie
  2. Tempe
  3. Tahu

Teks deskripsi

Menambahkan Video pada HTML

Elemen 'video' digunakan untuk menyisipkan video ke dalam halaman web. Elemen ini mendukung berbagai format video dan dapat dikontrol oleh pengguna (seperti memutar, menjeda, atau mengubah volume) menggunakan kontrol yang disediakan oleh browser.

Atribut-Atribut Penting:

src
Menentukan sumber file video yang akan ditampilkan
Contoh:'src="video.mp4"

controls
Menambahkan kontrol pemutar video (play, pause, volume) yang dapat digunakan oleh pengguna.
Contoh='controls'

widht dan height
Menentukan lebar dan tinggi tampilan video
Contoh:widht="600"' dan 'height="400"'

autoplay
Memutar video secara otomatis saat halaman dimuat
Contoh:'autoplay'

loop
Memutar video secara berulang-ulang tanpa henti
Contoh:'muted'

muted
Memulai video dalam keadaan tanpa suara
Contoh:'muted'

poster
Menentukan gambar yang ditampilkan sebelum video diputar
Contoh:'poster'

preload
Menentukan bagaimana browser harus melakukan pra-pemuatan video.
Nilai='none', 'metadata', 'auto'
Contoh='preload="auto"'
